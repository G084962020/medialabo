第3回　JavaScriptの基本

JavaScript は Web ページ上で複雑な機能を実現するプログラミング言語で、ページの自動更新やアニメーションなど動きのあるページを実現するためにJavascriptプログラムを使います。 HTMLでscript要素の中にJavascriptが書けます。console.logを使うとWeb開発ツールのコンソールにJavaのSystem.out.println()ように文字列などを出力することができます。 HTMLファイルとは別にJavascriptファイルを作成して、Javascriptファイルの内容をHTMLファイルに反映させることができる。 Javascriptでは計算もでき、変数を使った計算もしてくれる。 JavaとJavascriptでは言語の構文が違う。例えば、intはletで表現される。配列やオブジェクトをつかって、コンソールに表示したい情報だけを表示することができる。

第4回　DOMとJavaScript

DOMとはWebコンテンツを表す木構造のこと DOM を使用することによって、JavaScript などのプログラムによって、 Web コンテンツの内容を読み取ったり，コンテンツに変更を加えたりする処理が書けるようになる。 DOMはまずWeb開発ツールで編集する。その後にJavascriptの方に変更をそのまま加える。 Web開発ツールから表示されている文字を変えることができたり、CSSを変えたりすることができたり、文字を追加したり、削除したりできる。 今までは、コンソール上にしか、表示する方法を知らなかったが、画面上に表示できるようになった。
